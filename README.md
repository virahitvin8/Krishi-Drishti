# Krishi Drishti v3 — Complete Live App

**कृषि दृष्टि** — Satellite Vision for Smart Farming

This is the complete, production-ready version with:
- Exact brand colors and animated logo
- Full UI with individual percentages, expandable sections
- Transparent hotspot grid (Red = stressed, Bluish = healthy)
- All satellite analyses + DEM + Pest Risk
- Loading & processing animations
- Ready for Netlify + Render + Supabase deployment

## Quick Start

1. Push this folder to GitHub
2. Deploy `frontend` folder to Netlify
3. Deploy `backend` to Render (add your CDSE keys)
4. Set up Supabase (schema in deployment guide)
5. Your app is live 24/7

Full step-by-step guide is in `deployment/DEPLOYMENT_GUIDE.md`
