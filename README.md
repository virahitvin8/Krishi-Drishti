### Step-by-Step

1. **рҹ“Қ Select your land** вҖ” Draw on the interactive map, enter GPS coordinates, or upload a CSV file with multiple field locations
2. **рҹӣ°пёҸ Satellite scans your farm** вҖ” The system fetches the latest available imagery from Sentinel-2 (10m resolution), Sentinel-1 SAR (all-weather radar), and Landsat 8/9
3. **рҹ“Ҡ AI analyzes crop health** вҖ” Advanced vegetation indices are computed: NDVI (vigor), EVI (canopy), NDWI (water), REIP (early stress detection), SAVI (soil-adjusted), and GNDVI (chlorophyll)
4. **рҹ“Ӣ Get instant results** вҖ” Clear visual dashboard shows your overall health score, individual index breakdowns, weather data, pest risk assessment, and personalized recommendations
5. **рҹ“Ҳ Track over time** вҖ” Save your farms and monitor changes with periodic satellite passes. View trends via the integrated Grafana dashboard
