# Flutter Starter for Krishi Drishti

This folder contains a basic Flutter project structure to help you build a native mobile app in the future.

## Recommended Approach

**Phase 1 (Now):** Use the PWA version (already working).

**Phase 2:** Wrap the PWA using **Capacitor** or **PWABuilder** to generate APK quickly.

**Phase 3:** Build a proper Flutter app (this starter can be expanded).

## How to Use This Starter

1. Install Flutter SDK
2. Run:
   ```bash
   flutter create krishidrishti_app
   cd krishidrishti_app
   ```
3. Replace `lib/main.dart` with code that either:
   - Uses WebView to load your PWA, or
   - Rebuilds the UI natively in Flutter

## Next Steps
